# What Really Affects Developer Salaries?

## Motivation
Developer salaries vary widely across experience levels, education backgrounds,
and work arrangements. This project explores which factors are most strongly
associated with developer compensation using data from the 2024 Stack Overflow
Developer Survey. The goal is not to predict salaries perfectly, but to understand
the patterns and trends that influence developer pay.

## Questions of Interest
- How does professional coding experience relate to developer salaries?
- Do education level and remote work status influence compensation?
- What patterns emerge when comparing salary distribution across developers?

## Dataset
The dataset comes from the **2024 Stack Overflow Developer Survey**, which contains
responses from tens of thousands of developers worldwide. For this analysis, the
following variables were used:
- Annual compensation
- Years of professional coding experience
- Education level
- Remote work status

## Files in This Repository

This project follows the **CRISP-DM** process:
1. Business Understanding
2. Data Understanding and Exploratory Data Analysis

## Results Summary
The analysis shows that professional experience is strongly related to salary,
but it is not the only factor. Education level and remote work flexibility also
contribute to differences in compensation. Salary outcomes become more variable
at higher experience levels, indicating that multiple factors influence developer
pay.
